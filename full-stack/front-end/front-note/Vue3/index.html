# 下拉框 el-select

基本使用方法

v-for 里的 key 绑定的数据是唯一的，比如 id 就是唯一的

多选可设置 multiple 属性

如果使用 Select 的绑定值为对象类型，请务必指定 value-key 作为它的唯一性标识。

通过使用 value-key 属性，可以正确处理带有重复 label 的数据。这样虽然 label 是重复的，但可通过 id 来确认唯一性。
